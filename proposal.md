## In a paragraph or more, detail your project. What will your software do? What features will it have? How will it be executed?

Our project aims to create a portable oximeter that incorporates building both the  hardware and software aspect of the monitor. The monitor should measure the user's pulse and blood oxygen level in a quick and simple fashion by using a sensor that emits light into the user's fingertip. The light is captured by the photodetector and the sensor will determine the user's pulse and SPO2 levels which will be displayed nicely for the user to see via a digital screen. The montor should clipped onto the user's finger without creating any discomfort. To make the oximeter, we will first use 3D printing to make the outer shell, built the hardware portion, and then implement the software to ensure that it is a working gadget. 

Sophia as a pre-med and Salina as a Chemistry are both very interested in making healthcare more acessible. Thus, we were inspired by the simple yet highly effective mechanism of the oximeter in measuring a patient's pulse and SPO2 level. These type of monitors are widely used to quickly measure a patient's vital and thus, increases the patient-flow in a busy healthcare environment. Through this project, we wish to learn how a pulse oximeter work from start to finsh and hope to understand how modern technology improves healthcare.
